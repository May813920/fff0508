
# Sleep Posture Monitoring System

This project is a Streamlit-based real-time posture monitoring prototype for long-term care settings.

## Features
- Real-time webcam posture monitoring
- YOLOv8 pose estimation
- Detects:
  - Supine
  - Left-Side
  - Right-Side
  - No person
- Triggers an alert when the same posture is maintained too long
